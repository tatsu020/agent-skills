# Agent Skills

個人的に作成した agent skill 集です。

必要な skill だけを選んで使うことも、まとめて追加することもできます。

## Quickstart

```bash
npx skills add tatsu020/agent-skills
```

## Skills

### レビュー

- [`consult-chatgpt-pro`](skills/consult-chatgpt-pro): ChatGPT Pro を外部レビュアーとして使い、高リスクな技術判断や不確実な実装方針を批判的に確認します。返答はそのまま採用せず、ローカルの証拠と検証で使う内容を決めます。
